### Information on data files


##Count data

#Location and use
- data downloaded from paper supplement (saved in folder "paper_supplement") and from GEO server (saved in folder "geo")
- files contain RNAseq raw count data
- to be used for differential gene expression analysis

#Source (paper supplement)
https://doi.org/10.1016/j.gastha.2024.07.002

#Source (GEO)
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE268278



##Databases

#Location and use
- database used for gene-set enrichment analysis (GSEA) was downloaded from Broad institute remote server and deposited in folder "databases"

#Source (Broad institute server)
https://data.broadinstitute.org/gsea-msigdb/msigdb/release/2023.2.Mm/mh.all.v2023.2.Mm.symbols.gmt
